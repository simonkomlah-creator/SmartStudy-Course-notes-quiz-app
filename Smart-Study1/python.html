Variables and Data Types Variables are used to store values in Python.
Data types determine the type of value a variable can hold.
Understanding variables and data types is essential for writing effective Python code.

DATA TYPES IN PYTHON


1. Integers (int): whole numbers, e.g., 1, 2, 3.
Integers can be positive, negative, or zero.

2. Floats (float): decimal numbers, e.g., 3.14, -0.5.
Floats are used to represent numbers with fractional parts.

3. Strings (str): text, e.g., "hello", 'hello'.
Strings can be enclosed in single quotes or double quotes.

4. Boolean (bool): true or false.
Boolean values are used for conditional statements and logical operations.

5. List (list): ordered collection of values, e.g., [1, 2, 3].
Lists are mutable, meaning they can be modified after creation.

6. Tuple (tuple): ordered, immutable collection of values, e.g., (1, 2, 3).
Tuples are similar to lists but cannot be modified after creation.

7. Dictionary (dict): key-value pairs, e.g., {"name": "John", "age": 30}.
Dictionaries are used to store data in a structured way.

Exception Handling Exception handling is used to handle runtime errors that occur during the execution of a program.
Python provides several built-in exceptions that can be used to handle errors.
Try-Except Block:
The try-except block is used to handle exceptions.
The try block contains the code that might raise an exception,
and the except block contains the code that will be executed if an exception is raised.
try:
x = 5 / 0 except ZeroDivisionError:
print("Cannot divide by zero!")
Multiple Except Blocks:
You can use multiple except blocks to handle different types of exceptions.
try:
x = 5 / 0
except ZeroDivisionError:
print("Cannot divide by zero!")
except TypeError:
print("Invalid data type!")

Except Block with Alias:
You can use an alias to access the exception object.
try:
x = 5 / 0
except ZeroDivisionError as e:
print("Error:", e)

< Finally Block:
The finally block is used to execute code regardless of whether an exception was raised or not.
try:
x = 5 / 0
except ZeroDivisionError:
print("Cannot divide by zero!")
finally:
print("This code is always executed.")

< Raising Exceptions:
You can use the raise statement to raise an exception.
raise ValueError("Invalid value!")
Example Usage:
try:
x = 5 / 0
except ZeroDivisionError:
print("Cannot divide by zero!")
finally: print("This code is always executed.")
try:
raise ValueError("Invalid value!")
except ValueError as e:

Exception handling is used to handle runtime errors and make your code more robust.

List Comprehensions List comprehensions are a concise way to create lists in Python.
They are used to transform or filter existing lists.
Basic Syntax:
The basic syntax of a list comprehension is [expression for variable in iterable].
numbers = [1, 2, 3, 4, 5]
squares = [x**2 for x in numbers]
print(squares) # [1, 4, 9, 16, 25]

Filtering:
You can use an if statement to filter the elements of the list.
numbers = [1, 2, 3, 4, 5]
even_squares = [x**2 for x in numbers if x % 2 == 0]
print(even_squares) # [4, 16]

Multiple Loops:
You can use multiple loops to create a list comprehension.
numbers = [1, 2, 3]
letters = ['a', 'b', 'c']
pairs = [(x, y) for x in numbers for y in letters]
print(pairs) # [(1, 'a'), (1, 'b'), (1, 'c'), (2, 'a'), (2, 'b'), (2, 'c'), (3, 'a'), (3, 'b'), (3, 'c')]

Example Usage:
numbers = [1, 2, 3, 4, 5]
squares = [x**2 for x in numbers]
print(squares) # [1, 4, 9, 16, 25]
fruits = ['apple', 'banana', 'cherry']
uppercase_fruits = [fruit.upper() for fruit in fruits]
print(uppercase_fruits) # ['APPLE', 'BANANA', 'CHERRY']

List comprehensions are a concise way to create lists and are often used in Python programming.

Lambda Functions Lambda functions are small, anonymous functions that can be defined inline.
They are used to perform simple operations.
Basic Syntax:
The basic syntax of a lambda function is lambda arguments: expression.
add = lambda x, y: x + y
print(add(2, 3)) # 5

Use Cases:
Lambda functions are often used with functions like map(), filter(), and reduce().
numbers = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x**2, numbers))
print(squares) # [1, 4, 9, 16, 25]

Filtering:
Lambda functions can be used with filter() to filter elements from a list.
numbers = [1, 2, 3, 4, 5]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers) # [2, 4]

Sorting:
Lambda functions can be used as a key function for sorting.
fruits = ['apple', 'banana', 'cherry']
fruits.sort(key=lambda x: x.upper())
print(fruits) # ['apple', 'banana', 'cherry']

Example Usage:
add = lambda x, y: x + y
print(add(2, 3)) # 5
numbers = [1, 2, 3, 4, 5]
squares = list(map(lambda x: x**2, numbers))
print(squares) # [1, 4, 9, 16, 25]

Lambda functions are a concise way to define small, anonymous functions.

Regular Expressions Regular expressions (regex) are used to match patterns in strings.
Python provides the re module for working with regex.
Basic Syntax:
The basic syntax of a regex pattern is a string that defines the pattern to be matched.
import re
pattern = "hello"
string = "hello world"
match = re.search(pattern, string)
if match:
print("Match found!")

Special Characters:
Regex patterns use special characters to define the pattern.
import re
pattern = "^hello" # matches "hello" at the start of the string
string = "hello world"
match = re.search(pattern, string)
if match:
print("Match found!")

Common Patterns:
Some common regex patterns include:
  • . matches any character
  • ^ matches the start of the string
  • $ matches the end of the string
  • * matches zero or more occurrences
  • + matches one or more occurrences
  • ? matches zero or one occurrence
Example Usage:
import re
pattern = "\d+" # matches one or more digits
string = "hello 123 world"
match = re.search(pattern, string)
if match:
print("Match found:", match.group()) # 123

import re
pattern = "hello|world" # matches "hello" or "world"
string = "hello world"
matches = re.findall(pattern, string)
print(matches) # ['hello', 'world']

Regular expressions are a powerful tool for matching patterns in strings.

SciPy Library:
The SciPy library is used to perform scientific operations.
from scipy import integrate
result, error = integrate.quad(lambda x: x**2, 0, 1)
print(result)
